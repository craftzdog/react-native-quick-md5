# react-native-quick-md5

A native implementation of MD5

## Installation

```sh
npm install react-native-quick-md5
```

## Usage

```js
import QuickMd5 from "react-native-quick-md5";

// ...

const result = await QuickMd5.multiply(3, 7);
```
